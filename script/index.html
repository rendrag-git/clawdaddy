Frequently asked questions

What is OpenClaw?
OpenClaw is an open-source AI agent framework. This install script automates the entire setup process—Node.js, Chromium, VNC, firewall, systemd service, workspace—so you can have a running AI assistant in minutes instead of hours.
Do I need my own server?
Yes. Any Ubuntu VPS works—DigitalOcean, Hetzner, Linode, AWS, etc. A $5/month droplet with 1 GB RAM is enough to get started. The script handles all the server configuration for you.
What about the Anthropic API key?
You bring your own Anthropic API key and pay Anthropic directly for usage. This keeps your costs transparent and under your control. Most users spend $5–$20/month on API usage depending on how active their agent is.
Can I re-run the script?
Yes. The script is fully idempotent. It checks what's already installed, won't overwrite your config files without asking, and backs up existing configurations before making changes. Safe to run as many times as you need.
What if I need help?
Email support is included with your purchase. If you hit any issues during setup or configuration, we'll help you get running. Most issues are resolved within a few hours.
Is there a refund policy?
30-day money-back guarantee, no questions asked. If OpenClaw isn't for you, just email us and we'll refund you in full.
What's included in updates?
Lifetime access to script updates via the private GitHub repo. When we improve the installer or add new features, you get them automatically. Just pull the latest version and re-run.
